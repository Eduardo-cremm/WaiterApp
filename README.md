# WaiterApp

Desenvolvendo um aplicativo para restaurantes, focando na criação de um sistema de pedidos para garçons. Utilizaremos o Figma para o design e o Node.js com Express para o backend. Vamos configurar o TypeScript, criar um servidor, conectar ao MongoDB e definir rotas RESTful. Além disso, implementaremos models com Mongoose para gerenciar dados de categorias, produtos e pedidos. Ao final, teremos um aplicativo funcional que poderá ser usado em tablets por garçons, com funcionalidades como listagem de categorias, produtos e gerenciamento de pedidos.
